# Video Instructions

This folder contains videos for the Rosemary Dream property site.

## Hero Video

Place a file named `hero.mp4` in this folder to display it in the hero section.

### Requirements:
- Format: MP4 (H.264 codec recommended)
- Resolution: 1920x1080 or 1280x720
- Duration: 15-60 seconds recommended
- File size: Under 20MB for fast loading
- Style: Cinematic drone footage or property walkthrough

### Tips:
- Ensure the video loops smoothly
- Keep audio ambient or muted (the site mutes by default)
- Optimize with tools like HandBrake for web delivery

## Additional Videos

You can add additional videos (e.g., `tour.mp4`, `drone.mp4`) for the Video Tour section.
Update the VideoTour component to reference them as needed.
